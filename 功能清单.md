# AbaChat 功能清单

## 项目概述
AbaChat（阿巴聊天）是一个基于 WebSocket 的私密聊天室应用，支持一对一和多人聊天模式，通过房间口令实现访问控制。

## 技术架构
- **后端**: Node.js + WebSocket (ws) + HTTP 静态服务器
- **前端**: 原生 HTML/CSS/JavaScript，无框架依赖
- **通信**: WebSocket 实时双向通信

---

## 功能模块

### 1. 房间系统
| 功能 | 描述 |
|------|------|
| 房间创建 | 通过输入房间口令自动创建房间 |
| 访问控制 | 房间口令验证，保护隐私 |
| 聊天模式 | 支持一对一（最多2人）和多人聊天（最多10人）两种模式 |
| 空房间清理 | 房间无人时自动删除 |

### 2. 用户管理
| 功能 | 描述 |
|------|------|
| 昵称设置 | 可选昵称，留空则自动生成 |
| 昵称去重 | 重复昵称自动追加数字后缀（如 user_1） |
| 用户列表 | 实时显示房间内所有在线用户 |
| 加入/离开通知 | 用户进入或离开时显示系统消息 |

### 3. 消息功能
| 功能 | 描述 |
|------|------|
| 文本消息 | 支持最长 2000 字符的文本消息 |
| 消息时间戳 | 每条消息显示发送时间 |
| 消息类型 | 支持 text、emoji、image 三种消息类型 |
| 自动表情识别 | 单独发送单个 emoji 时自动识别类型 |
| XSS 防护 | 消息内容经过 HTML 转义处理 |
| 已读回执 | 可实时查看消息被阅读状态（✓✓ 已阅读） |

### 4. 实时交互
| 功能 | 描述 |
|------|------|
| 输入指示器 | 显示"XXX 正在输入..."提示，3秒后自动消失 |
| 在线状态 | 头部显示当前连接状态 |
| 心跳检测 | 30秒间隔保活检测，10秒超时断开 |

### 5. 界面功能
| 功能 | 描述 |
|------|------|
| 登录界面 | 模式选择、口令输入、昵称输入 |
| 等待界面 | 单人等待时显示等待动画和人數统计 |
| 聊天室界面 | 消息显示区、输入区、用户列表区 |
| 房间已满提示 | 房间满员时显示友好提示 |
| 表情面板 | 72个常用 emoji 表情，点击即发送 |
| 用户面板 | 可折叠的在线用户列表 |
| 响应式设计 | 适配移动端和桌面端 |

### 6. 浏览器适配
| 功能 | 描述 |
|------|------|
| 页面关闭提醒 | 离开页面时自动发送离开消息 |
| 回车发送 | 支持 Enter 键发送消息 |
| 表情点击发送 | 点击表情直接发送 |
| 自动重连 | 连接断开后提示用户 |

## 消息协议

### 客户端 → 服务器
| 消息类型 | 用途 |
|----------|------|
| `join` | 加入房间 |
| `chat_message` | 发送聊天消息 |
| `typing` | 发送输入中状态 |
| `stop_typing` | 停止输入状态 |
| `read_receipt` | 发送已读回执 |
| `leave` | 离开房间 |
| `ping` | 心跳检测 |

### 服务器 → 客户端
| 消息类型 | 用途 |
|----------|------|
| `join_success` | 加入成功 |
| `join_error` | 加入失败 |
| `user_joined` | 用户加入通知 |
| `user_left` | 用户离开通知 |
| `chat_message` | 收到聊天消息 |
| `typing` / `stop_typing` | 输入状态变化 |
| `read_receipt` | 已读回执 |
| `pong` | 心跳响应 |

## 配置参数
| 参数 | 默认值 | 说明 |
|------|--------|------|
| PORT | 3000 | 服务器端口 |
| MAX_MESSAGE_LENGTH | 2000 | 最大消息字符数 |
| MAX_MESSAGE_SIZE | 4096 | 最大消息体大小（字节） |
| TYPING_TIMEOUT | 3000 | 输入指示器超时（毫秒） |
| HEARTBEAT_INTERVAL | 30000 | 心跳间隔（毫秒） |
| HEARTBEAT_TIMEOUT | 10000 | 心跳超时（毫秒） |

---

## 依赖项
- `ws`: WebSocket 服务器
- `uuid`: 用户 ID 生成
